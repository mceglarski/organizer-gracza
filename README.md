# Organizer Gracza

This is an engineering thesis at the Polish-Japanese Academy of Information Technology. 

Demo: https://organizer-gracza.herokuapp.com/

# Requirments

dotnet version 5.0.4
node version 14.17.0
npm version 6.14.13

# Start backend

To start backend:
1) Open project
2) In terminal run:
   - dotnet ef database update
   - dotnet run
 
# Start frontend

To start frontend
1) Open project
2) In terminal run
   - npm install
   - ng serve
